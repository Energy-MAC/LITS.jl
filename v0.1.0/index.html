LITS.jl

LITS.jl is a Julia package for doing Power Systems Dynamic Modeling with Low Innertia Energy Sources.

Installation

You can install it by typing

julia> using Pkg
julia> Pkg.clone("https://github.com/Energy-MAC/LITS.jl.git")
 ```
 ## Usage

  Once installed, the `LITS` package can by used by typing

  ```julia
 using LITS
 ```
